## 3. Real Adapter 推荐接入顺序

1. 只读设备状态适配器：读取连接状态、版本、心跳、基础传感器，不下发运动或加液命令。
2. 扫码器只读接入：样本扫码、试剂扫码，保持人工确认和正式审计。
3. 温控/制冷只读接入：读取温度、连接状态、故障码，暂不闭环控制。
4. 液位、废液、纯水/PBS 状态只读接入：先作为预检输入，不自动解除阻断。
5. 命令 dry-run 层：生成命令但不驱动硬件，校验报文格式、幂等和审计。
6. 单模块受控写入：先低风险模块，每个模块独立工程二次认证和物理旁路确认。
7. 机械臂/加液/洗针/DAB 闭环：最后接入，需要完整硬件联锁、急停、夹具、防撞和人工确认 SOP。
