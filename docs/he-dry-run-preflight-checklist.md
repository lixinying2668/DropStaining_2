# HE 空载干跑前检查表

用途：真实硬件只读联调前，对 HE 空载干跑进行人工放行检查。此表不替代软件启动前预检，也不授权真实运动或加液；Real Adapter 未完成前仍 fail-closed。

正式现场放行和只读软件配置测试记录见：[`PRE-01 / PRE-02 现场只读测试用例`](../../stain_test/PRE-01-PRE-02现场只读测试用例.md)。

## A. 环境与数据

- [ ] 使用独立联调数据库或已备份正式数据库。
- [ ] 已确认当前环境不是 Production。
- [ ] `Device:Mode` 与本次目标一致。
- [ ] 执行器锁路径已指向本次联调目录。
- [ ] 安全日志目录已指向本次联调目录。
- [ ] 备份目录已指向本次联调目录。
- [ ] 最新 Migration 无待应用模型变更。

## B. 用户与权限

- [ ] 操作员账号可登录。
- [ ] 工程师账号可登录并可查看诊断页。
- [ ] 管理员账号可查看管理页。
- [ ] 操作员页面不显示工程技术细节。
- [ ] 工程写操作需要二次认证、原因和目标。

## C. HE 流程与样本

- [ ] 存在 Published 默认 HE 流程。
- [ ] HE 流程步骤、试剂需求和版本状态已由管理员确认。
- [ ] 创建 HE 通道前无活动运行占用。
- [ ] HE 通道选择后冻结 workflow snapshot。
- [ ] 空载干跑使用明确标识的测试样本/空载任务，不混入真实病理样本。

## D. 试剂与 DAB

- [ ] HE 所需试剂目录已启用。
- [ ] 最新试剂扫描无 INVALID 阻断项。
- [ ] HE 所需试剂瓶状态为 Available 且未过期。
- [ ] 试剂余量满足空载干跑要求。
- [ ] DAB 对 HE 非必需；如页面显示 DAB 状态，仅作为资源状态观察。

## E. 设备与安全

- [ ] Mock 模式下设备初始化通过。
- [ ] Real 只读联调时，不下发运动、加液、加热、扫码触发等动作命令。
- [ ] 废液、毒性废液、纯水、PBS、洗针等状态无阻断。
- [ ] 温控状态可读；Real 模式下不进行加热控制。
- [ ] 未存在未处理 Critical/Error 告警。

## F. 启动前预检

- [ ] `/api/run/preflight` 已执行。
- [ ] 预检报告 taskCount 与 HE 空载任务数一致。
- [ ] Reagents、Workflow、Device 相关阻断项已处理。
- [ ] 预检 state hash 只用于服务端校验，不在操作员页面暴露为技术细节。
- [ ] 若预检失败，停止干跑准备并回到对应页面处理。

## G. 放行记录

- [ ] 记录操作者、工程师、管理员姓名。
- [ ] 记录数据库、锁、日志、备份目录。
- [ ] 记录 HE 流程版本号。
- [ ] 记录预检时间和结果。
- [ ] 记录是否允许进入 Mock 空载干跑。
- [ ] 真实硬件只读联调不得记录为“已完成真实干跑”。
